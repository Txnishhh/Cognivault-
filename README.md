# 📚 Student Hub v1.0

**Student Hub** is a comprehensive Java console application designed to help students efficiently manage their academic and personal activities. It integrates task management, note-taking, scheduling, GPA tracking, expense management, and an intelligent study advisor into a single unified system.



## 🚀 Features

### 📋 Task Manager
- Add,delete,and manage tasks
- Mark tasks as completed
- View all or only pending tasks
- Smart reminders displayed on login

### 📝 Notes
- Create and store multi-line notes
- Simple input system using `END` terminator
- View and delete notes easily

###  Timetable / Schedule
- Manage weekly class schedule
- Filter schedule by specific day
- Add and remove class entries

###  GPA Calculator
- Calculates GPA using a 10-point grading scale
- Automatically assigns grades (A+, A, B, etc.)
- Supports weighted GPA based on credits
- Real-time GPA updates

| Marks | Grade | Points |
| ----- | ----- | ------ |
| 90+   | A+    | 10.0   |
| 80–89 | A     | 9.0    |
| 70–79 | B     | 8.0    |
| 60–69 | C     | 7.0    |
| 50–59 | D     | 6.0    |
| <50   | F     | 0.0    |


### 💸 Expense Tracker
- Record daily expenses with categories
- View total and category-wise spending
- Bill splitting feature for shared expenses

### 🤖 AI Study Advisor
- Analyzes tasks, GPA, and expenses
- Provides productivity and study suggestions
- Identifies weakest subject for improvement
- Recommends daily study hours

### ☁️ Data Export
- Displays structured JSON preview of user data
- Includes tasks, notes, GPA, and expenses
